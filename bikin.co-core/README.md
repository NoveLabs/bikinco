# Pijar Career Center

## Techstack
- Laravel
- MySQL
- Boostrap

## Requirements
```
PHP 7.3.3
Laravel Framework 7.28.4
```

## How to setup
```
- clone this repo
- create new database
- setup env
- composer install
- php artisan migrate
- php artisan db:seed
- open browser http://localhost:8000
```

## create apikey
```
run php artisan apikey:generate 'appname'
```
